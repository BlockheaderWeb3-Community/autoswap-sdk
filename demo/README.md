# AutoSwap SDK Demo

This demo project tests the published `autoswap-sdk` package from npm.

## Setup

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Set up environment variables:**

   ```bash
   cp env.sample .env
   ```

   Then edit `.env` with your actual credentials.

3. **Run the demo:**
   ```bash
   npm start
   ```
